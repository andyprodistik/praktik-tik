About HAXcms

HAXcms is a revolutionary content management system that combines the power of web components with the simplicity of static site generation. Built with modern web standards, HAXcms empowers creators to build beautiful, fast, and accessible websites without the complexity of traditional CMS platforms.

Our Mission

We believe that creating and managing web content should be:

  • Simple - No complex setup or configuration required
  • Fast - Lightning-fast performance with static generation
  • Accessible - Built with accessibility in mind from the ground up
  • Flexible - Customize and extend with web components
  • Open - Free and open-source for everyone

"HAXcms represents the future of content management - combining the best of static sites with the power of modern web components."

- HAXcms Team

Technology Stack

HAXcms is built on cutting-edge web technologies:

Web Components

Native browser support for reusable, encapsulated components that work everywhere.

JSON Outline Schema

A simple, powerful format for organizing and structuring your content.

Static Generation

Deploy anywhere with fast, secure, and scalable static files.
