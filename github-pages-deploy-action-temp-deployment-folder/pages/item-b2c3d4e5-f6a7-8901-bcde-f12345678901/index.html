Your First Steps

1. Edit Your Content

Click the "Edit page" button in the top bar to start editing. HAXcms provides a visual editor that makes content creation intuitive and fun.

2. Add New Pages

Use the page management tools to add new pages to your site. Organize them in a hierarchy that makes sense for your content.

3. Customize Your Theme

Choose from various themes or create your own. HAXcms themes are built with web components, making them highly customizable.

4. Insert Rich Content

Click "Merlin" to see all available content blocks:

  • Videos and media players
  • Interactive charts and graphs
  • Code samples with syntax highlighting
  • Image galleries and carousels
  • Quizzes and interactive elements
  • And much more!

Keyboard Shortcuts

Speed up your workflow with these handy keyboard shortcuts:

Action Shortcut
Save changes Ctrl/Cmd + S
Open Merlin (block picker) Ctrl/Cmd + M
Toggle edit mode Ctrl/Cmd + E
Create new page Ctrl/Cmd + N

Pro Tips

Tip: Press Enter in any paragraph to create a new one. Use the block picker (Merlin) to transform paragraphs into different content types.

Tip: All your content is saved as simple HTML files, making it easy to version control with Git and deploy anywhere.

Tip: HAXcms works great with GitHub Pages, Netlify, Vercel, and any static hosting service.
